# Apk-Akademik

Proyek aplikasi mobile akademik yang dibangun menggunakan Flutter Framework. Repositori ini dikembangkan untuk mengelola kebutuhan sistem informasi akademik berbasis mobile.

---

## 🚀 Panduan Memulai (Getting Started)

Ikuti instruksi di bawah ini untuk menduplikasi (*cloning*) proyek ini dan menjalankannya di lingkungan lokal Anda untuk proses pengembangan atau pengujian.

### 📋 Prasyarat (Prerequisites)

Sebelum melakukan instalasi, pastikan mesin lokal Anda sudah terpasang perangkat lunak berikut:

*   **Git** (Versi terbaru)
*   **Flutter SDK** (Pastikan *path* sudah terkonfigurasi di *environment variables*)
*   **Dart SDK** (Sudah termasuk saat menginstal Flutter)
*   **Android Studio / VS Code** (Lengkap dengan ekstensi Flutter & Dart)
*   **Emulator Android / Perangkat Fisik** (Dengan fitur USB Debugging aktif)

---
